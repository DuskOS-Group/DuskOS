[ LEGEND ]
S-UI: A server-based Graphical User Interface. (Can be console and Graphical)
D-UI: A desktop-based Graphical User Interface. (Can only be graphical, except in recovery mode.)

S-UI: will not have any graphical customization, except for desktop background and changing themes.
D-UI: will have full graphical customization including creating themes, functions, or events to be able to emulate other computers functionality, such as windows SNAP function on Vista+ systems.
